GETS/Data directory which contains the genomic data used in the tests. 

Data provided by David Koslicki. 

This corresponds to the 16S rRNA data used in the Quikr experiments of the paper  
D. Koslicki, S. F., G. Rosen, Quikr: a method for rapid reconstruction of bacterial communities via compressive sensing. 
Bioinformatics, 29/17, 2096--2102, 2013.
